# Artemis Landing Page Project Document

## Project Name

**Artemis** 

## One-Line Description

Artemis is a smart waste management platform that combines software, mobile access, and IoT-enabled trash bins to help organizations monitor waste in real time, automate reporting, and improve collection efficiency. 

## Tagline Options

- Platform powering the future generation of waste infrastructure. 
- Smart waste management for modern institutions and organizations.

***

## Overview

Artemis is a Waste Management Information System focused on helping governing organizations and institutions manage waste-related data through dashboards, data visualization, mobile tools, and IoT-connected trash bins.  The initial implementation context is Batangas State University, where Artemis is designed to monitor waste conditions, process waste management data, and support faster decision-making. 

The platform combines three core parts: a public/admin web app, a mobile app, and smart trash bins with IoT sensors.  Together, these components enable real-time bin monitoring, GPS-aware location visibility, waste data input, report generation, and more efficient waste collection workflows. 

***

## Problem Statement

Waste generation in the Philippines is rising, and institutions and local government units still struggle with effective waste management implementation.  The project materials state that waste generation was about 21.4M metric tons in 2020 and was projected to reach roughly 23M to 23.61M metric tons by 2025. 

Although RA 9003 requires structured waste management practices, implementation remains inconsistent, and many organizations still rely on manual processes that are costly, slow, and inefficient.  Existing systems often require significant human effort for data logging, monitoring, reporting, and collection planning. 

### Core pain points

- Manual waste data collection and reporting. 
- Poor real-time visibility into bin capacity and status. 
- Inefficient collection schedules that are not based on actual fill levels. 
- Difficulty generating compliance reports. 
- High effort required to implement and manage waste systems. 
- Limited monitoring and decision support for institutions and LGUs. 

***

## Solution

Artemis provides a centralized smart waste management ecosystem that allows organizations to monitor waste behavior anywhere and anytime, log waste management data easily, and generate reports on demand.  It also connects IoT-enabled smart bins to the platform so organizations can monitor fill level, location, and usage activity in real time. 

The system is built to reduce manual work, improve operational visibility, and support more efficient waste collection and planning.  Artemis also positions itself as a lower-cost and easier-to-adopt solution compared with traditional waste management modernization efforts.

## Product Components

### 1. Web Application

The web application is the main control center for organization-wide waste management.  It is intended for admins and institutional stakeholders who need analytics, reporting, and monitoring tools. 

#### Web app capabilities

- Dashboard view with data visualization and processed waste information. 
- Map view for monitoring trash bin status. 
- Data logs. 
- Admin controls, including granting admin access in mobile. 
- Historical data monitoring and decision support. 
- On-demand report generation. 

### 2. Mobile Application

The mobile app enables field-friendly access to Artemis features and allows easier data input and monitoring on the go. 

#### Mobile app capabilities

- Dashboard view with data visualization. 
- Map view for monitoring trash bins. 
- Data logs. 
- Data input tools. 
- Trash bin status viewing. 
- Guest view for students and faculty. 
- Dual language support: English and Tagalog. 
- Light mode and dark mode. 
- View nearest trash bin and its current status. 
- Mobile notifications related to collection needs. 

### 3. Smart Trash Bin / IoT Layer

The IoT component gives Artemis its real-time sensing and automation capability. 

#### IoT capabilities

- Capacity detection for fill-level monitoring. 
- Remote monitoring of bins. 
- Real-time analytics for collection optimization. 
- Real-time notifications when bins are full and need attention. 
- GPS-based location tracking. 
- Solar-powered operation. 
- Battery backup for continuity during outages or low sunlight. 
- Frequency-of-use logs sent in batches.

## Old Way vs New Way

Artemis reframes waste management from manual and reactive workflows into automated and real-time operations. 

### Operational shift

- Waste Collection 1.0: blindly scheduled collection. 
- Waste Collection 2.0: real-time status-based and pre-planned collection. 
- Data Visualization 1.0: manual entry, cleaning, exploration, and visualization. 
- Data Visualization 2.0: automated data cleaning and visualization with easier exploration. 
- Waste Management Report 1.0: manually generated and inefficient reports. 
- Waste Management Report 2.0: automated generated reports. 
- Recycling 1.0: manual sorting and extra facility investment. 
- Recycling 2.0: automated segregation using AI. 
- Data Forecasting 1.0: manual collection, analysis, training, and testing. 
- Data Forecasting 2.0: automated forecasting based on Artemis historical data.
